Preparation

Field Panel Connection Check

  • Turn off and close ZScaler
  • Disconnect from ABT Site
  • Confirm the computer IP address and subnet mask are on the same subnet intended for the field panel
  • Set up BacStac by going to C:\ProgramFiles(x86)\Cimetrics and running bacstac.bat
  • Set up a physical connection if one does not exist
  • Select the ethernet adaptor used to connect to the panel (the computer IP should autopopulate in the dropdown
  • Select "Attached"
  • Bacstac will be restarted automatically
  • unzip the CT database in your Job Folder
  • Open Job Editor
    • Open Online
    • Select the appropriate atom.fdb file for your job
  • Open System Profile
    • Insert new Insight
    • Name must match Computer name. Computer name can be found by opening command prompt (CMD) and typing "whoami" and pressing enter
    • Move the BLNs from the existing Insight into your Insight
    • Confirm the three names of the field panel (Name, System Name, Object Name), the panel instance ID, and the field panel IP Address ensuring that it is on the same subnet as your computer
  • Open MMI database transfer
    • Open your BLN on the left
    • Click the panel on the right
    • Direct Connect
    • Open Terminal Emulation via the "Tools" Menu
    • Enter the initial login and password ("high" for both) and change the password
    • Enter the characters 'S','H','E','S','M'
    • Enter the correct field panel name and IP address of the panel being mindful that the IP address of the field panel should be on the same subnet as the computer
  • Close CommTool completely
  • Open the job Online via Job Editor
  • Open System Profile
  • Panel should be connected

Functional Performance Test

Air System Basic Relinquish Default Check

  • Confirm via HMI the panel back-up and auto-restore configuration (On-site task)
  • Confirm all bacnet notification classes have been set up via System Profile (On-site task)
  • Confirm all engineering units are correct where inwc is used for pressure, pct is used for actuators, hz or pct is used for fan speed, %rh is used for humidity, ppm is used for CO2, ft3/m is used for cfm, s is used for seconds, m is used for minutes, and degF is used for temperature
  • Confirm all sensor slope & intercept values are correct keeping in mind that some 4-20ma sensors may require an isolated DC power supply such as the Kele DCP-1.5-W
  • Confirm relinquish defaults for all actuators and valves are set to 0%
  • Confirm relinquish defaults for all fans are set to 0% or 0 Hz
  • Confirm the following relinquish defaults for Static Control Loops:
    • Ramp Time: 300
    • Setpoint: 1.0
    • Reset High: 2.0
    • Reset Low: 1.0
    • P-Gain: 2000
    • I-Gain: 200
    • D-Gain: 0
    • Sample Time: 1
    • Bias: 0
    • Start: 0
  • Confirm the following relinquish defaults for Airflow Control Loops:
    • Ramp Time: 300
    • Setpoint: 1500
    • Reset High: 3000
    • Reset Low: 1500
    • P-Gain: 0
    • I-Gain: 4
    • D-Gain: 2
    • Sample Time: 1
    • Bias: 0
    • Start: 0
  • Confirm the following relinquish defaults for Temperature Control Loops:
    • Setpoint: 55
    • Reset High: 65
    • Reset Low: 55
    • P-Gain: 500
    • I-Gain: 50
    • D-Gain: 0
    • Sample Time: 1
    • Bias: 0
    • Start: 0
  • Confirm the following relinquish defaults for Humidity Control Loops:
    • Ramp Time: 300
    • Setpoint: 30
    • Reset High: 80
    • Reset Low: 30
    • P-Gain: 50
    • I-Gain: 5

  • Confirm all event enrollments for points have been set up with the correct triggers and times and that the appropriate notification classes have been assigned to alarmable points (On-site task)
  • Confirm all trends have been created using trend grid (found in trend wizard) and that notification class 7 has been assigned to all trended points (On-site task)
  • Confirm bacnet schedule (and corresponding command) has been set up. Occupancy times are correct and that all SSTO parameters have been set up if appropriate (On-site task)

Air System Basic Hardware Check

  • Using a magnehelic pressure guage and hand pump, ensure that all low and high static cutouts are tuned appropriately where low-static cutouts are typically tuned to -4"wc and high static cutouts at 4"wc
  • If the unit is equipped with a high-humidity cutout sensor, ensure the humidity sensor and cutout are an adequate distance from the humidifier to ensure they are not being sprayed directly
  • Confirm all Dampers and Actuators accept 0%, 50%, 100% commands where 0% is fully closed and 100% is fully open
  • If the unit is equipped with mixing dampers, confirm that economizer, exhaust and return dampers all modulate based on the 'MAD' or 'RAD' command where the mixing (return) damper actuates the opposite of the others
  • Command open the dampers to allow airflow to be drawn
  • Confirm all Fans go to minimum speed when started, accept half & full speed commands from the BMS as Hz or %
  • Confirm all Fans go to 0% or 0Hz when commanded to stop
  • Release all dampers and fan commands from Operator priority
Air Volume Tracking

When both supply and exhaust controls are present for the space, the equation for calculating volume differential offset (exhuast cfm setpoint - supply cfm setpoint) will be used. The terminal unit will maintain a differential cfm offset by executing the following sequence:

  • In conditions were the room volume differential offset is negative, the supply airflow setpoint will track the exhuast airflow setpoint to prevent contaminates from exiting the space.
  • In conditions were the room volume differential offset is positive, the exhaust airflow setpoint will track the supply airflow setpoint to prevent contaminates from entering the space.
  • In conditions were the room volume differential offset is neutral, the exhaust airflow setpoint will track the supply airflow setpoint to prevent contaminates from entering or exiting the space.
Dehumidification

The terminal unit will control humidity by executing the following sequence:

  • If the room humidity rises above the humidity setpoint by 5% (adjustable) the terminal unit will enter dehumidification mode and engage the dehumidification PID controller. Upon initialization, the airflow setpoint will be set to the minimum cooling cfm. If this fails to bring the room humidity down to setpoint, the airflow setpoint will be set to the maximum cooling cfm. As the humidity begins to fall, the airflow setpoint will revert to minimum cooling cfm before finally disengaging the cooling PID controller.
  • A signal will be sent to the Air Handling Unit that dehumidification is needed in a particular zone.
  • In the event that the room temperature falls below the heating setpoint during dehumidification, heating-mode will be initiated simulataneously and executed according to the VAV Heating Sequence.
Rapid Ventilation

The terminal unit will purge the space by executing the following sequence:

  • Utilizing the room operator unit(thermostat) the occupant will be able to manually override the terminal unit to maintain the maximum ventilation air volume setpoint for a period of 15 minutes (adjustable).
Greenleaf

The terminal unit will evaluate the energy efficiency of the room by analyzing the HVAC conditions of the space along with Lighting and Shading where applicable when utilizing Siemens Total Room Automation and execute the following sequence:

  • Occupants will receive a notification on the room operator unit indicating whether the conditions in the room are energy conditions are energy efficient. If the conditions are not efficient, the occupant will have the option to revert the room back to an energy efficient state.
Static Pressure Reset

Static Reset will be requested by the Central Controller by evaluating the following conditions from all terminal units served by a particular air handling unit every minute (adjustable):

  1. Damper Position is open more than 90% (adjustable).
  2. Airflow deviation from setpoint exceeds 0% (adjustable).
  3. The static pressure setpoint at the unit will increase by 0.1"wc (adjustable) every 5 minutes (adjustable) if 3 or more terminal units (adjustable) are requesting more pressure.
  4. The static pressure setpoint at the unit will decrease by 0.1"wc (adjustable) every 5 minutes (adjustable) if all units (adjustable) are satisfied.
  5. The terminal unit can be excluded from sending requests the the AHU through a manual command from the operator.
Temperature Reset

Temperature Reset will be requested by the Central Controller by evaluating the following conditions from all terminal units served by a particular air handling unit:

  1. The cooling demand percentage from all of the terminal units.
  2. The heating demand percentage from all of the terminal units.
  3. The highest 10 demands will be averaged by the Central Controller and will be used by the AHU to reset the discharge air temperature setpoint up and down.
  4. The terminal unit can be excluded from sending requests the the AHU through a manual command from the operator.
Operating Mode-Specific Functionality
Comfort(Occupied) Mode
  • Triggers:
    1. User defined schedule.
    2. Manual command from the central controller.
    3. Manual override via local room operator unit for 2 hours(adjustable).
    4. Automatic if a configured occupancy detector detects movement.
  • Sequence
    1. Terminal unit will maintain the occupied temperature setpoints.
    2. If the room temperature is within the heating/cooling deadband and room CO2 levels are below the high-limit threshold, the terminal unit will maintain the occupied minimum ventilation airflow setpoint. If room CO2 levels rise above 1000ppm (adjustable), the terminal unit will engage the ventilation PID controller to increase airflow into the space to meet the occupied maximum ventilation airflow setpoint until room CO2 levels fall below 800ppm (adjustable), at which point, the ventilation PID controller will disengage. The ventilation PID controller takes precedence over both the heating and cooling PID controllers.
    3. If the room temperature is within the heating/cooling deadband, the terminal unit will turn off.

    2. Manual command from the central controller.
    3. Automatic if a configured occupancy detector detects no movement for 10 minutes(adjustable).

    2. The ventilation PID controller takes precedence over both the heating and cooling PID controllers.
    3. If the room temperature is within the heating/cooling deadband, the terminal unit will turn off.
