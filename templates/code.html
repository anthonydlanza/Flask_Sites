This tool will help you find similar SOOs that have already been written and executed so you don't have to rewrite code that's most likely already been written. If you find a close enough match, click the result and you should get 1 of 3 things to speed up the programming process:

  • a .pcl file of the job that was already programmed
  • a link to the customer crsp site where you can search and rip the related code
  • The contact info for the programmer or specialist who executed the work previously

Directions for Comparison üë©‚Äçüíª

  1. Click the "Choose File" button and choose the SOO you would like to compare. This should be a .txt, .rtf, or .doc file
  2. Click the "Compare" button
  3. View the results and click on one of the SOOs that matches yours. A file will be downloaded
  4. Cross-check the SOO with yours. If a .pcl file is downloaded, test it in debugger and tweak it for your project

  3. Click the second "Choose File" button and choose the .pcl or .txt with contact info or crsp link
  4. Click the "Upload" button
